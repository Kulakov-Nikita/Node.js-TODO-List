### Лабораторная работа по созданию Web-приложения на React.js + Node.js

---

Адаптация раздела "Цитаты преподов" из группы "Подслушанно ВолгГТУ" в виде Web-приложения

Установка зависимостей:
```bash
npm install
```

Запуск:
```bash
npm run dev
```
